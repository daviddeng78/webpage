About This Project

This game is a trivia-style game, inspired by the popular mobile game Trivia Crack. Created by Team Trivia, David Deng and Ivan Yeung. 2021.

How the Code Works

Setup

The program sorts the text files that has the questions and answers in a certain format of a certain category and creates a dictionary. It will do this for every category and then will combine all of the dictionaries into a master dictionary. After it has finally created the dictionaries, it will prompt the player to type Trivia() to start the game.

The Game

The program will ask if the player wants to spin. If yes, it will choose randomly between the categories and pull a question from that category.
